# My Diet Coke Addiction - Version 2.0.2

## Added in V2.0.2

- Added Entry Time field under Entry Date
- Historical entries can now have their time edited
- Editing an entry loads the saved date and saved time
- Beverage caffeine now scales based on ounces:
  - Formula: logged ounces ÷ default ounces × default caffeine mg
- Caffeine values are rounded using standard math rules to 2 decimal places
- Rounded caffeine values are stored with the entry
- The caffeine field remains manually editable
- Added helper text: “Auto-calculated from beverage default. You can edit it.”
- Includes the V2.0.1 storage note explaining Safari vs Home Screen app storage

## Still Included
