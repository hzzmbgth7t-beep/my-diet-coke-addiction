# My Diet Coke Addiction — Version 1

This is the GitHub Pages version of the private iPhone-friendly soda tracker.

## What it does
- Log soda name, size, caffeinated yes/no, and clear yes/no
- Choose 8, 12, 16, or 20 oz
- Save soda names for future dropdown use
- Track daily ounces and 12 oz soda equivalents
- Set a daily limit
- Alert when the daily limit is reached
- Show charts by week, month, quarter, and year
- Export/import a JSON backup
- Use the custom Home Screen icon

## Put it on GitHub Pages
1. Create a free GitHub account if needed.
2. Create a new repository named `my-diet-coke-addiction`.
3. Upload every file and folder from this package into the repository.
4. In GitHub, open **Settings** → **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Click **Save**.
7. GitHub will create a Pages link. It may take a minute or two.

## Add it to your iPhone Home Screen
1. Open the GitHub Pages link in Safari on your iPhone.
2. Tap the Share button.
3. Tap **Add to Home Screen**.
4. Name it **My Diet Coke Addiction**.
5. Tap **Add**.

## Important
Your data is stored locally in Safari on your iPhone. Use **Export Backup** occasionally so you do not lose your logs if Safari data is cleared or you switch phones.
