# My Diet Coke Addiction

Private iPhone/iPad drink tracker web app.

## Features
- Custom icon support
- Totals at top
- Four quick ounce buttons
- Manual ounce entry
- Yes/No toggle controls
- Edit entries
- Delete entries
- Delete all entries
- Local device storage
- Add to Home Screen support

## Files
- index.html
- icon.png
- README.md

## Local Storage
All data stays on your device inside Safari local storage.
