# My Diet Coke Addiction

A private iPhone/iPad web app for tracking drinks, ounces, caffeine, and clear beverages directly in Safari using local storage.

No App Store required.  
No hosting required.  
No account required.  

---

# Features

## Version 1.1

- Track drink name
- Quick ounce buttons
  - 8 oz
  - 12 oz
  - 16 oz
  - 20 oz
  - 24 oz
  - 32 oz
- Manual ounce entry
- Caffeinated: Yes / No
- Clear: Yes / No
- Daily totals
- Edit saved entries
- Delete saved entries
- Delete all entries
- Local storage on device
- iPhone/iPad friendly interface
- Add to Home Screen support

# How To Use

## Open Locally

1. Download the project files
2. Open `index.html` in Safari
3. Use normally

OR

## Use With GitHub Pages

1. Upload `index.html` to a GitHub repository
2. Enable GitHub Pages
3. Open the generated website link in Safari

---

# Add To Home Screen (iPhone/iPad)

1. Open the app in Safari
2. Tap the Share button
3. Tap `Add to Home Screen`
4. Choose a name
5. Tap `Add`

The app will now behave similarly to a native app.

---

# Data Storage

All entries are stored locally in Safari using browser local storage.

No information is uploaded anywhere.

Deleting Safari website data will remove saved entries.

---

# Editing Entries

To edit an entry:

1. Tap `Edit`
2. Modify the information
3. Tap `Save Changes`

---

# Deleting Entries

## Delete One Entry
Tap `Delete`

## Delete All Entries
Tap `Delete All Entries`

---

# Known Limitations

- Data does not sync between devices
- Data can be lost if Safari website data is cleared
- No cloud backup yet
- No database integration yet
