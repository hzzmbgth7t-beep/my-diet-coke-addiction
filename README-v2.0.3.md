# My Diet Coke Addiction - Version 2.0.3

## Added in V2.0.3

- Added Entry Time field under Entry Date
- Historical entries can now have their saved time edited
- Editing an entry loads the original saved date and time
- Added Copy Entry button beside Edit/Delete
- Copy Entry duplicates an existing entry, opens it for editing, and defaults its time to now
- Beverage caffeine now scales by ounces:
  - logged ounces ÷ default ounces × default caffeine mg
- Caffeine values are rounded to 2 decimal places using standard math rules
- Rounded caffeine values are stored with the entry
- Caffeine field remains manually editable
- Reports now include:
  - Average Daily Caffeine
  - Highest Caffeine Day
