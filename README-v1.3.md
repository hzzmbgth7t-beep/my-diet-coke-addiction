# My Diet Coke Addiction — Version 1.3

Private iPhone/iPad drink tracker web app.

## Version 1.3 Updates

- Main page stays similar to the current layout
- Added a separate Reports area
- Added quick report buttons:
  - Yesterday
  - Last Week
  - Last Month
- Added custom report buttons:
  - Custom Day
  - Custom Week
  - Custom Month
- Added report totals:
  - Ounces
  - Entries
  - Caffeinated entries
- Added Drink Breakdown in reports
- Report entries are collapsed by default
- Today's Entries are collapsed by default
- Today's Entries only shows entries from the current day
- Older entries appear inside the appropriate report
- Older entries can be edited or deleted from Reports
- Editing an older entry preserves its original date and time
- Kept four quick ounce buttons:
  - 8 oz
  - 12 oz
  - 16 oz
  - 20 oz
- Kept manual ounce entry
- Kept Yes/No toggle buttons
- Kept local storage

## Data Storage

All data stays on your device inside Safari local storage.

Clearing Safari website data may delete saved entries.
