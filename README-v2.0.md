# My Diet Coke Addiction - Version 2.0 Migration-Safe

This version reads your current storage keys before migrating data:

- dietCokeTracker
- myDietCokeAddictionEntries

It also writes entries to the new V2 key:

- dietCokeEntriesV2

Your data is not stored inside this build. Your data remains in Safari/localStorage on your device. Open this from the same GitHub Pages URL to preserve the existing local storage.

## New in V2.0

- Home screen beverage buttons
- 2 columns x 3 rows of saved beverage slots
- Quick Entry button
- Add Beverage buttons
- Tap beverage to log
- Long-press beverage to edit
- Beverage setup with image, initials, default ounces, caffeine mg, carbonated, and clear
- Selected-date totals on entry screen
- Reports with caffeine mg, carbonated count, clear count, drink breakdown
- Backup/import data tools

## Install

Unzip and upload the individual files to GitHub:

- index.html
- README-v2.0.md

Keep your existing icon as `icon.png`.

Do not upload the ZIP itself.
